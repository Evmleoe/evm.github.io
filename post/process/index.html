Linux / Ubuntu 进程命令

Posted on 2023-07-26，2 min read

📃记录 Ubuntu 系统进程常用指令。

一、 ps -ef 命令

1、查询运行进程

ps -ef

2、查询名称带有“find”的所有进程

ps -ef | grep find

3、查询并过滤进程id

ps -ef  grep find  grep -v grep  cut -c 9-15

4、查询并过滤进程id，并杀死该进程

ps -ef l grep find  grep -v grep l cut -c 9-15  xargs kill -9

如果权限不足的话，尝试加上sudo

ps -ef  grep find  grep -v grep  cut -c 9-15 xargs sudo kill -9

说明:
grep find的输出结果是，所有含有关键字“find”的进程；
grep -v grep是在列出的进程中去除含有关键字“grep”的进程；
cut-c 9-15是截取输入行的第9个字符到第15个字符，而这正好是进程号PID；
xargsQ kill -s 9中的xargs命令是用来把前面命令的输出结果 (PID) 作为kill -s 9命令的参数，并执行该命令。kill -s 9会强行杀掉指定进程。

二、 ps -aux 命令

ps -aux 是显示所有进程和其状态
其余的检索方式跟 ps -ef 差不多，可以直接将 ps -ef 命令替换成 ps -aux 。

三、 kill 命令

1、kill
作用: 根据进程号杀死进程
用法:

kill [信号代码] [进程PID]

注意: kill -9 来强制终止退出
特殊用法:
停止一个进程，而并不消灭这个进程

kill -STOP [PID]

重新开始一个停止的进程

kill -CONT [PID]

强迫进程立即停止，并且不实施清理操作

kill -KILL[PID]

终止你拥有的全部进程

kill -9 -1

2、killall
作用: 通过程序的名字，直接杀死所有进程
用法:

killall [正在运行的程序名]

例如：

killall -9 mysql

3、pkill
作用: 通过程序的名字，直接杀死所有进程
用法:

pkill [正在运行的程序名]

如

pkill mysql
